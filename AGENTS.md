# Gleam for Python Programmers

This tutorial is an introduction to programming in Gleam for
programmers who are comfortable working with Python, but have never
used a strongly-typed or pure functional language before, and have not
used Erlang. Learners are more interested in useful applications than
in computer science theory.

Read ~/.ai/mccole.md for a description of the mccole static site
generator used to format these lessons.

Each lesson is in its own subdirectory with its own ./subdir/src
directory holding Gleam source code and its own gleam.toml file.
Each lesson subdirectory may hold many short examples that can be run
separately from one another.

Write point-form bullets rather than full prose.

*NEVER* use em-dashes (or "--" or "---" in place of em-dashes).
